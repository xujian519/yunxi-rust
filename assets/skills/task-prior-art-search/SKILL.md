# 任务1_2：现有技术调研与对比分析

> **场景**: 专利撰写

## 任务概述

**任务名称**: 现有技术调研与对比分析

**任务目标**:
- 基于平台数据资产进行现有技术检索
- 从向量库、知识图谱、专利数据库检索相关技术
- 识别最接近的现有技术
- 分析本发明与现有技术的区别
- 评估新颖性和创造性
- 为权利要求布局提供依据

**输入材料**:
- TASK_1_1的技术方案理解报告
- 提取的核心技术特征
- 发明人提供的参考文献（如有）

**输出成果**:
- 现有技术调研报告
- 对比分析表
- 新颖性/创造性评估
- 权利要求布局建议
