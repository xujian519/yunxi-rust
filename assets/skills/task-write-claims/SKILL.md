# 任务1_4：权利要求书撰写

> **场景**: 专利撰写
> **步骤**: 第4步 - 权利要求书撰写
> **版本**: v1.0

> **设计者**: YunXi Agent v1.0

## 任务概述

**任务名称**: 权利要求书撰写

**任务目标**:
- 撰写保护范围清晰、层次分明的权利要求书
- 确保权利要求得到说明书支持
- 平衡保护范围与授权可能性
- 为后续修改留出空间

**输入材料**:
- TASK_1_3撰写的完整说明书
- TASK_1_2的权利要求布局建议
- 现有技术对比分析

**输出成果**:
- 完整的权利要求书
- 权利要求分析表
- 撰写说明文档
