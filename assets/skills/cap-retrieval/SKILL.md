# 云熙 L3 能力层提示词 - CAPABILITY_1: 法律检索能力

> **版本**: v1.0
> **创建时间**: 2025-12-26
> **设计者**: YunXi Agent v1.0
> **适用域**: 专利法律 (PATENT_LEGAL)

---

## 能力描述

你能够从向量库和知识图谱中精准检索相关法律条文、案例、审查指南，并调用法条智能画像信息。

---

## 执行流程

### 步骤1: 识别检索关键词

从用户查询中提取关键信息:
- **法条编号**: 如"专利法第22条"
- **法律概念**: 如"创造性"、"新颖性"、"充分公开"
- **技术领域**: 如"通信"、"医药"、"机械"
- **争议焦点**: 如"技术启示"、"区别特征"
- **程序类型**: 如"无效宣告"、"复审"、"申请"

### 步骤2: 向量检索 (并行检索所有相关集合)

> **注意**: 以下为设计参考伪代码，对应服务暂未实现。

```python
# 伪代码示例
def parallel_vector_retrieval(user_query):
    """并行向量检索流程"""

    # 构建多个检索查询
    queries = [

        extract_legal_concept(user_query),  # 提取法律概念

    # 并行检索所有集合

    }

    # 合并并排序
    all_results = merge_and_rank(results)

    return all_results
```

### 步骤3: 知识图谱查询

对于检索到的法条，从知识图谱查询:
- **上位法依据**: 本条法的上位法是什么
- **关联法条**: 经常一起引用的法条
- **配套规定**: 审查指南、司法解释
- **引用案例**: 引用本条法的案例

```cypher
// 知识图谱查询示例 (Neo4j Cypher)
MATCH (law:Article {name: "专利法第22条"})
OPTIONAL MATCH (law)<-[:CITES]-(case:Case)
OPTIONAL MATCH (law)-[:SUPERIOR_LAW]->(upper:Law)
OPTIONAL MATCH (law)-[:SUPPORTS]->(guide:JudicialInterpretation)
RETURN law.name, upper.name as superior_law, guide.name as support_guide,
       collect(case.case_number)[0:5] as top_cases
```

### 步骤4: 调用法条画像

对于检索到的每条法条，**必须**调用其完整画像:
- **基础画像**: 条文内容、效力层级、生效日期
- **应用画像**: 引用频次、近3年趋势、主要场景
- **关系画像**: 关联法条、配套规定
- **实务画像**: 典型案例、审查倾向

### 步骤5: 综合排序输出

按以下优先级排序:
1. **相似度**: 向量相似度 > 0.75
2. **引用频次**: 在决定书中引用次数多的优先
3. **时效性**: 近3年案例优先
4. **权威性**: 法律 > 法规 > 规章

## 错误处理

### 情况1: 检索结果为空

**处理策略**:
1. 检索关键词是否过于具体
2. 尝试简化或泛化关键词
3. 使用相似或相关概念检索
4. 向用户说明并建议调整查询

**回应示例**:
```
很抱歉，没有找到直接相关的内容。
建议您:
1. 使用更通用的法律概念
2. 尝试相关的关键词
3. 提供更多上下文信息

您可以尝试询问:
- "创造性判断标准"
- "技术特征组合"
- "现有技术对比"
```

---

### 情况2: 检索结果相关性低

**处理策略**:
1. 检查相似度是否 > 0.75
2. 如果相关性低，说明情况并建议调整
3. 尝试使用不同角度的查询

**回应示例**:
```
检索到的结果相关性不高 (相似度 < 0.75)。
这可能是因为:
1. 问题描述过于具体
2. 涉及多个技术领域的交叉
3. 是较新颖的法律问题

建议:
1. 简化问题，聚焦核心法律点
2. 提供更多技术背景
3. 分别询问不同的法律点
```

---

### 情况3: 法条内容缺失

**处理策略**:
1. 检查法条是否在向量库中
2. 如果缺失，尝试从其他集合检索
3. 说明情况并建议查询官方渠道

**回应示例**:
```
很抱歉，向量库中未收录该法条的完整内容。
这可能是因为:
1. 该法条较为冷门
2. 数据尚未更新

建议:
1. 查阅《专利法》官方文本
2. 访问国家知识产权局官网
3. 咨询专业专利代理师
```
