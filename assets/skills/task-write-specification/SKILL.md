# 任务1_3：说明书撰写

> **场景**: 专利撰写
> **步骤**: 第3步 - 说明书撰写
> **版本**: v1.0
> **创建时间**: 2025-12-26
> **设计者**: YunXi Agent v1.0

---

## 任务概述

**任务名称**: 说明书撰写

**任务目标**:
- 撰写完整、规范的专利说明书
- 清楚、完整地公开技术方案
- 突出创新点和区别特征
- 为权利要求提供充分支持

**输入材料**:
- TASK_1_1的技术方案理解报告
- TASK_1_2的现有技术调研报告
- 技术交底书和补充材料

**输出成果**:
- 完整的专利说明书
- 说明书附图（如有）
- 摘要初稿

## 执行流程

### 第一步：撰写规划

**🤝 人机交互点1：确认撰写规划**

```
【云熙】用户，我准备开始撰写说明书，首先确认撰写规划。

📋 说明书结构规划：

┌─────────────────────────────────────────────────────────┐
│  说明书标准结构                                           │
│                                                           │
│  一、发明名称                                              │
│  - 简明扼要，全面反映技术主题                             │
│  - 建议名称: {{基于技术方案给出建议}}                      │
│                                                           │
│  二、技术领域                                              │
│  - 明确本发明所属或应用的技术领域                         │
│  - 建议字数: 50-100字                                     │
│                                                           │
│  三、背景技术                                              │
│  - 描述现有技术状况                                       │
│  - 指出存在的问题和缺陷                                   │
│  - 建议字数: 300-500字                                    │
│  - 重点: 客观描述D1、D2的不足                            │
│                                                           │
│  四、发明内容                                              │
│  - 要解决的技术问题 (1-2个核心问题)                       │
│  - 技术方案 (详细描述)                                    │
│  - 技术效果 (详细说明)                                    │
│  - 建议字数: 800-1500字                                   │
│  - 重点: 突出区别特征和优势                               │
│                                                           │
│  五、具体实施方式                                          │
│  - 实施例1 (详细描述)                                     │
│  - 实施例2 (变形方案)                                     │
│  - 实施例3 (可选，如适用)                                 │
│  - 建议字数: 1500-3000字                                  │
│  - 重点: 充分公开技术细节                                 │
│                                                           │
│  六、附图说明                                              │
│  - 简要说明各附图                                         │
│                                                           │
│  七、摘要                                                  │
│  - 简要说明技术方案                                       │
│  - 建议字数: 300字左右                                    │
│                                                           │
│  预计总字数: 3000-5000字                                  │
└─────────────────────────────────────────────────────────┘

📊 撰写策略：

基于现有技术分析，采用以下策略：

1. 背景技术部分:
   ✅ 客观描述D1、D2的技术方案
   ✅ 指出其存在的具体问题
   ✅ 为本申请的创新做铺垫

2. 发明内容部分:
   ✅ 清晰表述技术问题
   ✅ 详细描述技术方案（突出区别特征）
   ✅ 充分说明技术效果

3. 具体实施方式部分:
   ✅ 提供多个实施例
   ✅ 详细描述技术细节
   ✅ 覆盖权利要求的所有特征

🤝 请您确认：
1. 这个结构是否合适？
2. 字数安排是否合理？
3. 是否需要调整各部分比重？

【您可以选择】：
A. 确认，开始撰写
B. 调整规划（请说明）
C. 我有其他要求
```
