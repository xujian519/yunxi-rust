## 执行流程

### 第一步：撰写说明书摘要

#### 摘要撰写规范

```
摘要撰写要求：

┌─────────────────────────────────────────────────────────┐
│  一、基本要求                                             │
│                                                           │
│  字数要求: 一般不超过300字                                │
│  内容要求: 简要说明发明的技术领域、技术方案、技术效果     │
│  语言要求: 简明扼要，客观准确                             │
│                                                           │
│  二、必须包含的内容                                       │
│                                                           │
│  ✅ 发明所属的技术领域                                    │
│  ✅ 技术方案的核心内容                                    │
│  ✅ 主要技术效果                                           │
│  ✅ （如有）最优实施方式的关键点                          │
│                                                           │
│  三、不应包含的内容                                       │
│                                                           │
│  ❌ 商业性宣传用语                                        │
│  ❌ 与发明无关的说明                                      │
│  ❌ 过于抽象空泛的描述                                    │
│  ❌ 推测性的效果描述                                      │
│                                                           │
│  四、撰写技巧                                             │
│                                                           │
│  1. 开头: 直接点明技术领域和主题                          │
│  2. 中间: 简述技术方案的核心创新点                        │
│  3. 结尾: 说明主要技术效果                                │
│  4. 用词: 使用专业、准确、简洁的词语                      │
│  5. 句式: 多用陈述句，避免疑问句和感叹句                  │
└─────────────────────────────────────────────────────────┘
```
